# manual-testing-opencart
Manual Testing Project – OpenCart E-commerce Application

<h2 align="left">🔍 Project Overview</h2>
<p>
This project demonstrates manual testing of an e-commerce web application (OpenCart). It includes complete testing artifacts such as test plan, test cases, bug reports, and execution reports.
</p>

<h2 align="left">🎯 Objectives</h2>

* Validate core functionalities of the application.
* Identify defects and usability issues.
* Ensure system meets business requirements.
 
<h2 align="left">🛠️ Tools Used</h2>

* Microsoft Excel
* Google Sheets
* Browser DevTools

<h2 align="left">📂 Project Artifacts</h2>

| Document | Description |
|----------|--------------|
| Test Plan | Testing strategy and scope |
| Test Cases | Detailed test scenarios |
| Bug Report | List of identified defects |
